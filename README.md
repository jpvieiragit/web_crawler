Simple Crawler API

## Dependencies

I recommend you use [pipenv](https://github.com/pypa/pipenv)

```
pipenv install
pipenv shell
```

## Testing our API

You're now ready to test the API.
```
flask run
```
More details about endpoints access [documentation](https://app.swaggerhub.com/apis-docs/jpvieiragit/PontoTel/1.0.0).
